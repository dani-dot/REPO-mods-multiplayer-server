# A guide to make your modded REPO multiplayer experience better (works for windows and linux)
This guide uses awc21 's method , files and instuctions.
If u want more information read his guide:
https://github.com/awc21/BepInEx-PhotonRedir

# Instructions
1. Go to https://github.com/dani-dot/Games-for-me/blob/main/repo.txt and open one of the links (doesnt matter which one)
2. Download and extract the .zip file in the location u want the game to be
3. Download r2modman ( this is a mod manager .exe for windows and .AppImage for linux ): https://r2modman.com/download-v3-2-9/ and install it
4. Download the modpack 
5. Open r2modman, search for repo and at Profile Selection tab press Import/Update , press From file and search for moduri_1770737820333.r2z (this is the modpack you dowloaded)
